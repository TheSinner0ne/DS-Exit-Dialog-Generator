# DS-Exit-Dialog-Generator
This is Simple App, intended to help DroidScript user, especially users, focus on making HTML Apps. This Generator is simple tool to create a Exit Dialog which you can design it, in a way you like, though it may seems limited in some way.

## Installing The App
You can use this app in two ways, *cloning the git* or installing the App, you can download in the Release Page section. This app can run offline.

## How to build this into APK
This app was made using the DroidScript App, available in Google Play Store. Its the tool you can use,though you nedd premium account in order to create a apk or abb file.
